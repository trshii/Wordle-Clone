# Wordle-Clone
## Progression of Wordle Clone Creation

Welcome to my Wordle Clone Repository, where I'll be storing my wordle clone project progressions.

If you wish to play, I'll be adding instructions on each version on how to install the necessary libraries and how to run the game itself.

## wordle_clone_v1_tmnl (Terminal Game)

1. Install Requirements
    - run `pip install -r requirements.txt` on your terminal
    - be sure that you are in the correct directory (wordle_clone_v1_tmnl)

2. Run `main.py`

## wordle_clone_v2_web (Webpage Game)

1. Install Requirements
    - run `pip install -r requirements.txt` on your terminal
    - be sure that you are in the correct directory (wordle_clone_v2_web)

2. Run `app.py`

3. Open the indicated link where the game is running
